Calgary Secondary Suite Map

The purpose of this website is provide a simple way to search and display data provided by City of Calgary (available to the public at data.calgary.ca) to easily find approved Secondary Suites in Calgary and whether they were approved under the current code or using an alternative solution the City of Calgary have made available for Secondary Suites built without a permit prior to March 12, 2018.

How to use
Navaigate to the  map. Find a pin for the location of the suite you would like to see information on. When clicking on the pin if secondary suite is labeled as "NEW" it conforms with current code in force at the time of the application issue date, if the secondary suite is labeled as "EXISTING" it conforms to an alternative solution which the city has changed slightly over the course of the existing suite program slowly moving towards what current code requirements are.
